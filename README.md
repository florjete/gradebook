## Setup

1. Clone the repository:

```bash
git clone https://github.com/florjete/gradebook-cli.git
cd gradebook-cli
```

2. Create and activate a virtual environment:

```bash
python -m venv venv
# Windows (PowerShell)
venv\Scripts\activate
# macOS/Linux
source venv/bin/activate
```
